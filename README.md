## 📋 What is ux-editorjs?

ux-editorjs is a simple tool to help you create and manage content in blocks. It lets you write using small sections called blocks, then saves your work in clean JSON format. This format makes it easy to move, edit, or use your content in other apps or websites.

This tool works well with popular frameworks like Symfony and comes bundled with features to work smoothly with AssetMapper, Webpack Encore, and EasyAdmin. You do not need to know coding to use it.

## 🖥️ System Requirements

Before you start, make sure your computer has:

- Windows 10 or newer (64-bit preferred)
- At least 4 GB of RAM
- 1 GHz or faster processor
- 100 MB of free disk space
- A modern web browser (Edge, Chrome, Firefox, or Safari)

No additional setup is needed beyond installing the program.
